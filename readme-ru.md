## Для запуска

Скачиваем все модули, зависимости и компилируем

```
$ npm i && npm run build
```

Запускаем сервер:

```
$ npm start
```

Во время разработки используйте команду `watch`:

```
$ npm run watch
```

[All npm commands](docs/npm.md)
